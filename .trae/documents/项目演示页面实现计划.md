# 项目演示页面实现计划

## 目标
在个人博客网站中添加项目演示功能，包括：
1. 在 index.html 左上角添加"项目演示"按钮
2. 创建新的项目演示页面 demo.html
3. 实现点击卡片播放视频功能

## 实现步骤

### 1. 修改 index.html - 添加"项目演示"按钮
- 在左下角添加"项目演示"按钮（与右上角"AI模式"按钮对称）
- 使用与 小技能按钮相似的样式
- 点击后跳转到 demo.html

### 2. 创建 demo.html 项目演示页面
- 页面结构参考 index.html，保留相同的背景和样式
- 创建项目卡片网格布局
- 第一个项目卡片：抖音客服系统自动化
  - 包含图标、标题、描述
  - 点击卡片后打开视频播放器（模态框形式）

### 3. 创建或复用 CSS 样式
- 在 style.css 中添加演示页面需要的样式
- 复用现有的按钮样式和卡片样式
- 添加视频播放器模态框样式

### 4. 创建视频播放器功能
- 点击项目卡片时，显示视频模态框
- 模态框包含视频播放器（HTML5 video 标签）
- 模态框包含关闭按钮
- 点击模态框背景可关闭

## 文件清单
- `index.html` - 添加项目演示按钮
- `demo.html` - 新建项目演示页面
- `css/style.css` - 添加演示页面相关样式
