# 页面过渡动画实现计划

## 目标
为 index.html 和 AI.html 之间的页面跳转添加炫酷的过渡动画效果。

## 现有代码结构分析
- **style.css**: 主样式文件，包含主题变量（青色#00d4ff、粉色#ff00aa、紫色#7b2cbf等霓虹色调）
- **AI.css**: AI页面专属样式
- **main.js**: 主JavaScript文件
- 已有星空背景canvas动画、loading屏幕等效果

## 动画设计方案

### 方案：星际传送门效果

#### 1. 进入动画（index → AI）
- **触发点**: 点击 "AI模式" 按钮
- **效果描述**:
  - 按钮位置产生一个扩展的光圈（青色→粉色渐变）
  - 光圈迅速扩大形成圆形遮罩
  - 遮罩内部产生粒子漩涡效果（模拟星际传送）
  - 遮罩完全覆盖屏幕后执行页面跳转

#### 2. 返回动画（AI → index）
- **触发点**: 点击 "返回" 按钮
- **效果描述**:
  - 屏幕从边缘向中心收缩（反向传送效果）
  - 中心出现光芒扩散
  - 带有拖尾效果的粒子从四周向中心汇聚
  - 完成后执行页面跳转

## 实现步骤

### Step 1: 创建过渡动画样式文件
- 文件名: `css/transition.css`
- 包含:
  - `.page-transition-overlay` - 全屏遮罩层样式
  - `.portal-circle` - 传送门圆形效果
  - `.particle` - 粒子效果样式
  - 各阶段动画keyframes定义

### Step 2: 修改 HTML 结构
- 在 index.html 的 `<body>` 末尾添加过渡遮罩层div
- 在 AI.html 的 `<body>` 末尾添加过渡遮罩层div

### Step 3: 创建 JavaScript 过渡控制逻辑
- 文件名: `script/transition.js`（新文件）
- 包含:
  - `initTransition()` - 初始化过渡效果
  - `playEnterAI()` - 进入AI页面的动画
  - `playReturnIndex()` - 返回index页面的动画
  - `createParticles()` - 创建粒子效果
  - 动画完成后的回调处理

### Step 4: 在各页面引入新文件
- 修改 index.html: 添加 transition.css 引用，添加 transition.js 引用
- 修改 AI.html: 添加 transition.css 引用，添加 transition.js 引用
- 为 "AI模式" 按钮添加点击事件监听
- 为 "返回" 按钮添加点击事件监听

## 动画详细技术点

### 核心CSS动画
```css
/* 传送门扩展 */
@keyframes portalExpand {
    0% { transform: scale(0); opacity: 1; }
    50% { transform: scale(2); opacity: 0.8; }
    100% { transform: scale(50); opacity: 1; }
}

/* 粒子旋转 */
@keyframes particleRotate {
    0% { transform: rotate(0deg) translateX(50px); opacity: 1; }
    100% { transform: rotate(360deg) translateX(50px); opacity: 0; }
}

/* 反向收缩 */
@keyframes portalContract {
    0% { transform: scale(50); opacity: 1; }
    50% { transform: scale(2); opacity: 0.8; }
    100% { transform: scale(0); opacity: 1; }
}
```

### JavaScript控制逻辑
- 使用 `setTimeout` 控制动画时序
- 在动画50%处执行 `window.location.href` 跳转
- 动态创建和移除DOM元素管理粒子效果

## 预期效果
- 进入AI: 青色传送门从按钮位置扩展，伴随粉色粒子旋转，屏幕被光圈覆盖后跳转
- 返回首页: 屏幕从外向内收缩，中心汇聚成亮点后跳转
- 整个过程约 1.2 秒，流畅且具有科技感
